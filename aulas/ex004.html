Operadores (Parte1)


11- Operadores (Parte1) - Curso JavaScript #07


Operadores:
    aritméticos
    atribuição
    relacionais
    lógicos
    ternário

------------------------------------------------

Aritméticos: 
    5 +  2 = 7
    5 -  2 = 3
    5 *  2 = 10
    5 /  2 = 2.5
    5 %  2 = 1 (resto da divisão)
    5 ** 2 = 25 (potencia)

------------------------------------------------

Ordem de Precedência:
    ()
    **
    *   /   %   
    +   -
( Os da mesma linha, fazer conforme aparecer, da esquerda para direita)

------------------------------------------------

Exemplos de precedência:
    5 + 3 / 2 = 6.5 (4 nao!) 5 + (3 / 2)

    var a = 5 + 3           == 8
    var b = a % 5           == 3
    var c = 5 * b ** 2      == 45
    var d = 10 - a / 2      == 6
    var e = 6 * 2 / d       == 2
    var f = b % e + 4 / e   == 3

------------------------------------------------

Auto-atribuição:            Simplificando:
var n = 3       n == 3
n = n + 4       n == 7      n += 4
n = n - 5       n == 2      n -= 5
n = n * 4       n == 8      n *- 4
n = n / 2       n == 4      n /= 2
n = n ** 2      n == 16     n **= 2
n = n % 5       n == 1      n %= 5

------------------------------------------------

Incremento:
    var x = 5
    x = x + 1       x == 6      x++ ou ++x (+1)
    x = x - 1       x == 5      x-- ou --x (-1)

Operadores (Parte 2)


12- Operadores (Parte 2) - Curso JavaScript #08


Relacionais:
        5 >  2   true
        7 <  4   false
        8 >= 8   true
        9 <= 7   false
        5 == 5  true
        4 != 4  false 
------------------------------------------------
Identidade em JS:
        5 == 5      true
        5 == '5'    true
        5 === '5'   false
        5 === 5     true
        5 !== '5'   true
------------------------------------------------
Lógicos:
        !   negação     (nao / not)
        &&  conjunção   (e / and)
        ||  disjunção   (ou / or)

        true ! false
        false ! true 

        true && true    true 
        true && false   false
        false && true   false
        false && false  false

        true || true    true 
        true || false   true 
        false || true   true 
        false || false  false
------------------------------------------------

Precedência de tipos de operadores:
        aritméticos
        relacionais (nao tem ordem de precedência)
        lógicos
        
Precedência dos operadores lógicos:
        ! 
        && 
        ||

------------------------------------------------
Exemplos:
idade >= 15 && idade <= 17    // a idade esta entre 15 e 17?
estado == 'RJ' || estado == 'SP'    // o estado e Rio ou Sao Paulo?
salario > 1500 && sexo != 'M'       // o salario e 1500 e nao e homem?

------------------------------------------------

Ternário: ? :

    teste ? true : false

    teste   teste logico
    true    se for verdadeiro
    false   se for falso

    media >= 7.0 ? "Aprovado" : "Reprovado"
